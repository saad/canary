# Canary

The canary catches the change before it hits you. An autonomous
regulatory-intelligence agent for a credit union's risk/compliance unit:
monitors the Federal Register, NCUA, NACHA, and IRS; auto-opens a case for
every detected change; classifies impact and owning business units; drafts
the owner notification; delivers it and advances LOW/MED cases autonomously;
escalates HIGH ones to a human through an audited autonomy gate — every case
tracked NEW → TRIAGED → NOTIFIED → IN_PROGRESS → CLOSED with an actor-labeled
timeline. The case workspace UI is generated by Thesys C1 from a custom
component library; facts always render live from the database (the model
composes, it never owns the facts).

## Setup

Prerequisites: Python 3.11+ with [uv](https://docs.astral.sh/uv/). Node is
only needed if you change the frontend (`ui/dist` is committed).

```sh
git clone <this repo> && cd Canary
uv sync
cp .env.example .env     # fill in what you have — see below
```

Every integration is optional and env-flagged; a missing credential degrades
to a local fallback and never breaks the run:

| Key(s) | Powers | Without it |
|---|---|---|
| AWS credentials (boto3 default chain) + `AWS_REGION` | Bedrock classification + notification drafting | keyword-heuristic classifier, template drafts |
| `THESYS_API_KEY` | C1-generated case workspace | locally composed workspace, same components |
| `CLICKHOUSE_*` | audit/brief sink + audit-tab readback | SQLite only |
| `LANGFUSE_*` | tracing for every LLM + C1 call | no traces |
| `GUILDAI_*` | control-plane review of escalations (needs your own published Guild agent) | skipped |
| `GOOGLE_MAPS_API_KEY` | field-of-membership geocoding | Nominatim |
| `SMTP_*` + `NOTIFY_EMAIL_TO` | real notification delivery | drafts recorded, delivery skipped |

## Run

```sh
DEMO_POLL_SECONDS=5 ./scripts/demo_reset.sh   # fresh DB → real items cased via the live pipeline → server
# → open http://127.0.0.1:8400
uv run python -m harness.redetect             # the demo keypress: forget one real rule; watch it re-detected live
```

The reset backfills the queue by running the freshest *real* documents from
the live sources through the full pipeline, then pre-composes every C1
workspace (clicks open in ~30ms). `redetect` erases the agent's memory of one
real NACHA rule so the next poll re-detects it live — classify, escalate,
draft, all on real data. Console narration prints the timing numbers:
`⏱ keypress → escalation visible` and `⏱ keypress → draft attached`.

UI rebuild after frontend changes: `cd ui && npm install && npx vite build`
(the Python server serves `ui/dist`; `/legacy` keeps the old single-file
dashboard).

Other entry points:

```sh
uv run python -m harness.poller --interval 15   # console-only autonomy mode (Enter to acknowledge)
uv run python -m harness.cases                  # print the case queue + timelines
uv run python -m agents.fom "301 W 2nd St, Austin TX"          # FOM evidence chain
uv run python -m agents.member_agent "Can we serve a member at …?"
uv run python -m harness.run_step1               # adapter smoke test
uv run python -m harness.run_step2 --auto-ack    # end-to-end pipeline test
```

## Layout

- `sources/` — adapters, one interface: `fetch_latest() -> list[Item]`.
  Live Federal Register API; NCUA/NACHA/IRS scrapes with committed snapshot
  fallbacks (refreshed on every successful live fetch); `injector.py` keeps
  the synthetic-item path for offline testing.
- `agents/` — Bedrock classifier + notification drafter (Langfuse-traced via
  one `llm.converse` chokepoint), FOM/member agents.
- `harness/` — SQLite state, polling loop, autonomy gate, case lifecycle
  (`cases.py`), real delivery (`notify.py`), demo tooling (`backfill.py`,
  `redetect.py`, `seed.py`), ClickHouse sink, demo server.
- `ui/` — React + Tailwind three-pane case manager; the right-panel workspace
  is composed by Thesys C1 from four custom components (Zod-schema'd), with
  live-data overrides and a local fallback (`C1_DISABLED=1`).
- `config.py` / `.env.example` — all integrations env-flagged with fallbacks.
